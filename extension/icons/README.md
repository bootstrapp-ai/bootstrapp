# Extension Icons

Place your extension icons here:

- `icon16.png` - 16x16 pixels (toolbar, favicon)
- `icon48.png` - 48x48 pixels (extensions page)
- `icon128.png` - 128x128 pixels (Chrome Web Store)

## Quick Icon Generation

You can generate simple icons using this SVG as a base:

```svg
<svg xmlns="http://www.w3.org/2000/svg" width="128" height="128" viewBox="0 0 128 128">
  <rect width="128" height="128" rx="16" fill="#fbbf24"/>
  <text x="64" y="88" font-family="Arial, sans-serif" font-size="72" font-weight="bold" fill="#1a1a1a" text-anchor="middle">B</text>
</svg>
```

Or use any icon generation tool to create PNG versions.
