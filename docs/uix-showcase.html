Getting Started

To use @bootstrapp/uix in your project, add the importmap and import the components you need:

<script type="importmap"> { "imports": { "lit-html": "https://esm.sh/lit-html", "@bootstrapp/types": "https://esm.sh/@bootstrapp/types@0.1.0", "@bootstrapp/view": "https://esm.sh/@bootstrapp/view@0.1.0", "@bootstrapp/uix": "https://esm.sh/@bootstrapp/uix@0.1.0" } } </script> <link rel="stylesheet" href="https://esm.sh/@bootstrapp/uix@0.1.0/theme.css"> <script type="module"> // Import specific components import '@bootstrapp/uix/display/button.js'; import '@bootstrapp/uix/layout/card.js'; import '@bootstrapp/uix/form/input.js'; </script> <!-- Use components in your HTML --> <uix-card> <h3 slot="header">Welcome</h3> <p>Start building beautiful UIs!</p> <div slot="footer"> <uix-button primary>Get Started</uix-button> </div> </uix-card>
