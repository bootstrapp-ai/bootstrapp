@bootstrapp/view Examples

Basic component examples demonstrating the View.define() API

Basic Counter Component

A simple reactive counter demonstrating properties, methods, and event handling.

View.define('my-counter', { properties: { count: T.number({ defaultValue: 0 }), label: T.string({ defaultValue: 'Counter' }) }, increment() { this.count++; }, render() { return html` <h2>${this.label}</h2> <p>Count: ${this.count}</p> <button @click=${this.increment}>Increment</button> `; } });

Form Component

A form component demonstrating input handling and custom events.

View.define('my-form', { properties: { name: T.string({ defaultValue: '' }), message: T.string({ defaultValue: '' }) }, _handleSubmit(e) { e.preventDefault(); this.emit('form-submit', { name: this.name, message: this.message }); alert(`Submitted: ${this.name} - ${this.message}`); }, render() { return html` <form @submit=${this._handleSubmit}> <input type="text" placeholder="Name" .value=${this.name} @input=${e => this.name = e.target.value} > <textarea placeholder="Message" .value=${this.message} @input=${e => this.message = e.target.value} ></textarea> <button type="submit">Submit</button> </form> `; } });

Getting Started

To use @bootstrapp/view in your project, add the importmap and start defining components:

<script type="importmap"> { "imports": { "lit-html": "https://esm.sh/lit-html", "@bootstrapp/types": "https://esm.sh/@bootstrapp/types@0.1.0", "@bootstrapp/view": "https://esm.sh/@bootstrapp/view@0.1.0" } } </script> <script type="module"> import View from '@bootstrapp/view'; import { html } from 'lit'; import T from '@bootstrapp/types'; View.define('my-component', { // Component definition }); </script>
